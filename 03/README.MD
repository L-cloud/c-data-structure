## 리스트 추상 자료형

| 이름            | 입력                          |  출력       |
| :-------------- | :---------------------------: | ----------: |
| createList()    | 최대 원소 갯수 : n            | 리스트 l    |
| deleteList()    | 리스트 l                      |  N/A        |
| isFull()        | N/A                           | 성공/실패   |
| addElement()    | 리스트 l, 원소 위치 p, 원소 e | 성공/실패   |
| removeElement() | 리스트 l, 원소 위치 p         | 성공/실패   |
| clearList()     | 리스트 l                      | N/A         |
| getListLength() | N/A                           | 원소의 갯수 |
| getElement()    | 리스트 l, 원소 위치 p         | 원소 e      |
