## 큐 추상 자료형

| 이름            | 입력                          |  출력       |
| :-------------- | :---------------------------- | :---------- |
| createQueue()   | 최대 원소 갯수 : n            | 리스트 l    |
| enqueue()       | 리스트 l, 원소 e              | 성공/실패   |
| dequeue()       | 리스트 l                      | 원소 e      |
| peek()          | 리스트 l                      | 원소 e      |
| deleteQueue()   | 리스트 l                      | N/A         |
| isStackFull()   | 리스트 l                      | 원소의 갯수 |
| isStackEmpty()  | 리스트 l                      | 성공/실패   |
