## 이진 트리 추상 자료형

| 이름               | 입력              |  출력     |
| :----------------- | :---------------- | :-------- |
| createTree()       |                   | 트리 l    |
| deleteTree()       |                   | N/A       |
| getRoot()          |                   | 노드 n    |
| addLeftChild()     |  노드 n, 노드 c   | 성공/실패 |
| addRightChild()    |  노드 n, 노드 c   | 성공/실패 |
| getLeftChild()     |  노드 n           | 노드 c    |
| getRightChild()    |  노드 n           | 노드 c    |
| removeLeftChild()  |  노드 n           | 성공/실패 |
| removeRightChild() |  노드 n           | 성공/실패 |
