## 그래프 추상 자료형

| 이름               | 입력                |  출력     |
| :----------------- | :------------------ | :-------- |
| createGraph()      |                     | 그래프 g  |
| deleteGraph()      | 그래프 g            | N/A       |
| addVertex()        | 그래프 g, Vertext v | 성공/실패 |
| removeVertex()     | 그래프 g, Vertext v | 성공/실패 |
| addEdge()          | 그래프 g, Edge e    | 성공/실패 |
| removeEdge()       | 그래프 g, Edge e    | 성공/실패 |
